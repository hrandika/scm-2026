git branch🔹 1. What is Git?
Git is a version control system
It helps track changes in code
Allows multiple developers to work together
🔹 2. Basic Git Commands
✅ git init
Create a new Git repository
Used at the start of a project

👉 Example:

git init
✅ git clone
Copy an existing repository from GitHub

👉 Example:

git clone https://github.com/user/repo.git
✅ git status
Shows current state of files
Tells:
Modified files
Staged files
Untracked files

👉 Example:

git status
✅ git add
Add files to staging area

👉 Example:

git add file.txt
git add .
✅ git commit
Save changes permanently

👉 Example:

git commit -m "Added login feature"
✅ git log
Shows commit history

👉 Example:

git log
🔹 3. Branching Commands
✅ git branch
List or create branches

👉 Example:

git branch
git branch new-feature
✅ git checkout
Switch to another branch

👉 Example:

git checkout new-feature
✅ git switch (modern way)
Easier way to switch branches

👉 Example:

git switch new-feature
✅ git merge
Combine branches

👉 Example:

git merge new-feature
🔹 4. Remote Repository Commands
✅ git remote add
Connect local repo to GitHub

👉 Example:

git remote add origin https://github.com/user/repo.git
✅ git push
Upload code to GitHub

👉 Example:

git push origin main
✅ git pull
Download latest changes

👉 Example:

git pull origin main
✅ git fetch
Get updates without merging

👉 Example:

git fetch
🔹 5. Undo Commands
✅ git reset
Undo commits or staging

👉 Example:

git reset HEAD file.txt
✅ git revert
Undo a commit safely (recommended)

👉 Example:

git revert commit_id
✅ git stash
Save changes temporarily

👉 Example:

git stash
git stash pop
🔹 6. Tagging
✅ git tag
Mark important versions

👉 Example:

git tag v1.0
🔹 7. Config Command
✅ git config
Set user details

👉 Example:

git config --global user.name "Pathum"
git config --global user.email "email@example.com"
⚡ Common Workflow (Very Important for Exams)
git init
git add .
git commit -m "Initial commit"
git branch feature
git checkout feature
git add .
git commit -m "New feature"
git checkout main
git merge feature
git push origin main
