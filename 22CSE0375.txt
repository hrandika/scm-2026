Git Note 

Git is a distributed version control system used to track changes in files and collaborate with other developers. It allows multiple people to work on the same project without overwriting each other's work.

In Git, a repository (repo) is used to store project files and their history. Developers can create branches to work on new features without affecting the main project. After completing the work, the changes can be merged back into the main branch.

Common Git commands:

git clone – Copy a repository from GitHub to your computer
git branch – Create or view branches
git checkout – Switch between branches
git add – Add files to staging area
git commit – Save changes with a message
git push – Upload changes to GitHub
git pull – Download latest changes from GitHub

Git is widely used in software development for version control, collaboration, and project management.
